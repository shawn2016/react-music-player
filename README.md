# react-music-player

技术栈
`react16` `react-router4` `webpack4` `jquery.jplayer` `jquery` `es6`

#### Description

使用React构建一款音乐播放器

#### Download

```
git clone https://github.com/shawn2016/react-music-player.git
```

#### Install

```
npm install
```

#### Run

```
npm start
```
